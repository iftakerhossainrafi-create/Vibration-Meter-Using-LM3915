# Vibration-Meter-Using-LM3915
Overview

A vibration monitoring system using a piezoelectric sensor and LM3915 LED driver IC. The system converts mechanical vibrations into electrical signals and displays vibration intensity through a 10-level LED bar graph.

Features
1. Real-time vibration detection
2. 10-level LED indication
3. Analog signal visualization
4. Low-cost implementation
5. Easy sensor interfacing

Components Used
1. LM3915 IC
2. Piezoelectric Sensor
3. LEDs (10)
4. Resistors
5. Capacitors
6. Power Supply

Working Principle

The piezoelectric sensor generates voltage proportional to mechanical vibration. The LM3915 processes this signal and drives LEDs according to signal amplitude, providing a visual representation of vibration intensity.

Applications
1. Machine condition monitoring
2. Vibration analysis
3. Educational instrumentation projects
4. Industrial monitoring
